Inleiding Programmeren

De Plantenverzoeker

Individuele opdracht

Periode van het vak: 5 weken

De opdracht

De opdracht was om een eigen versie te maken van een tamagotchi. Met HTML, CSS en Javascript.

Proces

In de werkgroepen kregen we korte opdrachten dat mogelijk ook gebruikt zou kunnen worden in jouw eigen eindopdracht. Binnen dit vak was het leerdoel om Javascript te leren kennen. Ik heb een website gebouwd waarmee je een plant kan verzorgen. Door het te verzorgen groeit het uit tot een bloem.

Reflectie

De moeilijkheid lag natuurlijk in het schrijven van Javascript code, maar ik ben blij dat ik er opgegeven moment veel van snapte en werkende code heb kunnen schrijven.
